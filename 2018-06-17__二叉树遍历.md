## 二叉树遍历

### 递归写法

#### 前序遍历

根结点 ---> 左子树 ---> 右子树

```js
function preOrder (root) {
  if (!root) {
    return
  }
  console.log(root.value)
  preOrder(root.left)
  preOrder(root.right)
}
```

#### 中序遍历

左子树---> 根结点 ---> 右子树

```js
function inOrder (root) {
  if (!root) {
    return
  }
  inOrder(root.left)
  console.log(root.value)
  inOrder(root.right)
}
```

#### 后序遍历

左子树 ---> 右子树 ---> 根结点

```js
function postOrder (root) {
  if (!root) {
    return
  }
  postOrder(root.left)
  postOrder(root.right)
  console.log(root.value)
}
```
